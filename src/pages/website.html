The guestbook

Last updated on: {{ 2022-02-20 | prettyDate }}

Currently the guestbook is showing entries submitted via Webmention. Webmentions are then picked up by Brid.gy, so in theory it should pick up some mentions from social medias like Twitter and Mastodon.

My Webmentions are currently possible because of Webmention.io. I've been debating for a while and I decided to request my Webmentions on demand (by this I mean requesting them via Javascript on page load). There's a lot of people who have created super cool solutions to cache and save Webmentions for Eleventy but for now this is my preferred solution. This is mostly because in the end of the day this is a static website and I don't build my website frequently enough to get the latest status of my Webmentions. Also time and data storage concerns.

In the future, I would like my guestbook to also allow form submission using a similar proccess shared in the CSS Tricks article "JAMstack Comments".
